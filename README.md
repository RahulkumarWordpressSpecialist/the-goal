# The Goal - Coaching Institute

A modern, responsive landing page for "The Goal" coaching institute located in Samastipur, Bihar. This landing page is designed to provide information about the coaching institute to prospective students of Class 11 and 12 (Arts & Science).

## Features

- **Responsive Design**: Works seamlessly on all device sizes
- **Modern UI**: Clean and professional appearance
- **Interactive Elements**: Smooth scrolling, mobile menu toggle
- **Contact Form**: JavaScript-based form validation
- **Informative Sections**: About, Courses, Why Choose Us, Contact

## Sections

1. **Header**: Institute name, tagline, and navigation menu
2. **Hero**: Welcome message with call-to-action
3. **About**: Brief description of the institute
4. **Courses**: Details about Class 11 & 12 courses (Arts & Science)
5. **Why Choose Us**: Key benefits of joining the institute
6. **Contact**: Address, phone, email, and contact form
7. **Footer**: Copyright information

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)
- Font Awesome Icons

## How to Use

1. Simply open the `index.html` file in any modern web browser
2. No server required - all functionality works client-side
